API Pokémon Primera Generación

Una API RESTful con información de los 151 Pokémon originales

Bienvenido

Esta API proporciona información detallada sobre los Pokémon de la primera generación (Kanto). Puedes acceder a los datos de Pokémon, sus tipos y realizar diversas operaciones.

Endpoints Disponibles

Autenticación

POST /api/auth/register - Registrar nuevo usuario
{ "username": "tu_usuario", "password": "tu_contraseña" }
POST /api/auth/login - Iniciar sesión y obtener token JWT
{ "username": "tu_usuario", "password": "tu_contraseña" }

Pokémon

GET /api/pokemon - Obtener todos los Pokémon (requiere autenticación)
GET /api/pokemon/:id - Obtener un Pokémon por ID (requiere autenticación)
POST /api/seed - Transferir la base de datos de todos los tipos por nombre y por ID (requiere autenticación)
POST /api/pokemon - Crear un nuevo Pokémon (requiere autenticación)
PUT /api/pokemon/:id - Actualizar un Pokémon (requiere autenticación)
DELETE /api/pokemon/:id - Eliminar un Pokémon (requiere autenticación)
GET /api/pokemon/name/:name - Buscar Pokémon por nombre (requiere autenticación)
GET /api/pokemon/filter/type?type=electric - Filtrar Pokémon por tipo (requiere autenticación)
GET /api/pokemon/filter/legendary - Filtrar Pokémon legendarios (requiere autenticación)

Tipos

GET /api/types - Obtener todos los tipos de Pokémon (requiere autenticación)
GET /api/types/:id - Obtener un tipo por ID (requiere autenticación)
